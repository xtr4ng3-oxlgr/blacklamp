# Security Policy

BLACKLAMP es una herramienta defensiva para revisión local.

## Alcance

BLACKLAMP puede analizar enlaces de forma local, revisar carpetas elegidas por el usuario, calcular hashes, mover archivos seleccionados a cuarentena, revisar entradas de inicio en modo lectura y generar reportes locales.

BLACKLAMP no debe incorporar explotación, fuerza bruta, keylogging, monitoreo oculto, exfiltración, evasión, persistencia, borrado automático ni telemetría obligatoria.

## Datos

Los datos se mantienen en la máquina del usuario. Los reportes se guardan localmente. La herramienta no envía información a servicios externos.
