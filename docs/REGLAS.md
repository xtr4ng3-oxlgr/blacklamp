# Reglas defensivas

BLACKLAMP usa reglas locales.

No reemplaza antivirus. No consulta reputación online. No determina culpabilidad. Ayuda a frenar antes de abrir algo raro.

Categorías principales:

- enlaces sospechosos,
- archivos de riesgo,
- dobles extensiones,
- inicio sospechoso,
- comandos de shell,
- rutas sensibles,
- nombres engañosos.
